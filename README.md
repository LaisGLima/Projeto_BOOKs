# Projeto BOOks - Gerenciador de bibliotecas
## Projeto para desktop em Java para modernizar e simplificar a gestão de bibliotecas, com foco em segurança, catalogação de livros, automação de aluguel/devolução e facilitação de pagamentos de multas.

### Linguagem Java e SQL

### Atividade proposta pelo Prof.Ildo Vieira - Disciplina-Programação orientada a objetos em Java - Estácio-Parangaba
